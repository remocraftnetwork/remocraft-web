Reglas del Servidor

Las infracciones en el servidor están clasificadas en tres categorías según su gravedad. Las sanciones se aplicarán en función de la seriedad de cada infracción.

  1. Grave
    • Uso de hacks: cualquier modificación no autorizada para obtener ventajas injustas, como X-ray, fly, velocidad, etc. Si quieres usar un cliente o mod específico, consulta primero con el staff.
    • Uso de autoclicker para PvP: el uso de autoclicker solo se permite para farmeo.
    • Explotación de bugs o exploits: aprovechar errores del juego para obtener una ventaja clara sobre otros jugadores.
    • Lenguaje ofensivo extremo: comentarios que atacan directamente la salud, identidad o situaciones personales de otros jugadores, como desear la muerte a alguien.
  2. Medio
    • Actitud tóxica: conductas que, sin ser infracciones directas de PvP, afectan negativamente a otros jugadores.
    • Lenguaje ofensivo: insultos o lenguaje inadecuado sin llegar a ataques extremos.
    • Construcciones inadecuadas: estructuras que afectan el rendimiento del servidor o son molestas para otros jugadores.
    • Hacerse pasar por admin/staff: no admitimos nombres similares a los del owner, administradores o mods. El jugador tendrá 30 días para cambiarse el nombre; de no hacerlo, podrá ser baneado hasta nuevo aviso.
    • Skins o nombres inapropiados: contenido ofensivo, sexualmente explícito, político o inapropiado.
  3. Bajo
    • Spam: mensajes repetitivos en el chat que afectan la conversación.
    • Flood excesivo: enviar mensajes o emojis de forma continua en el chat.
    • Uso constante de mayúsculas: escribir de forma excesiva en mayúsculas.
    • Hablar de otros servidores: promocionar o discutir otros servidores dentro del chat o Discord.

¿Qué se considera actitud tóxica?

Se considera actitud tóxica cualquier comportamiento que moleste o afecte repetidamente a otros jugadores, aunque no infrinja directamente las reglas de PvP.

  1. Acoso repetido: perseguir o atacar constantemente al mismo usuario para impedirle jugar normalmente.
  2. Provocaciones después de una muerte: burlarse o reírse de forma descarada del jugador vencido, especialmente si se hace repetidamente.
  3. Acoso por parte del jugador que murió: responder con insultos, provocaciones o persecución constante.
  4. Estafas: engaños para que otro jugador entregue objetos, dinero o recursos valiosos.
  5. TPA Kill: usar solicitudes de teletransporte para engañar a un jugador y matarlo inesperadamente.
  6. Desconectarse reiteradamente en PvP: desconectarse varias veces para evitar el combate.

Comprar rangos VIP no te exime de cumplir las reglas. Si rompes una regla, recibirás la misma sanción que cualquier otro usuario.
