¿Quiénes somos?

RemoCraft nació con la visión de ofrecer una experiencia Survival RPG profunda y respetuosa. Buscamos crear una comunidad donde cada jugador pueda progresar a través de sus habilidades, fundar naciones poderosas y explorar un mapa diseñado a mano lleno de secretos y dungeons. Aquí, tu experiencia no es solo un nivel, es el maná que alimenta tus poderes.

Preguntas frecuentes.

Preguntas comunes sobre RemoCraft. Si quieres añadir más, dínosla y la añadiremos.

¿El servidor es premium?
Sí — el servidor requiere una cuenta Minecraft Java (premium) para unirse.
¿Se puede jugar desde Bedrock?
No — actualmente solo soportamos Minecraft Java Edition (cuentas premium).
¿Es el servidor pay-to-win?
No — no diseñamos mecánicas pay-to-win. Si crees que algo lo es, infórmanos y lo revisaremos.
